# VULCAN
*Chemical kinetics python code for exoplanetary atmospheres*

VULCAN is constructed for gaseous chemistry from 500 to 2500 K using a reduced C- H-O chemical network with about 300 reactions. At this moment, VULCAN is under the final stages of development, i.e. wrapping up various modules, adding documentation and cleaning up the chemistry data. The first test version will soon be realeased in *August, 2016*. 

The Exoclime Simulation Platform ([ESP][1]) develops a set of open-source codes
for research on exoplanets. The three parts of the ESP are
  - [HELIOS][2] radiative transfer and retrieval,
  - [THOR][3] atmospheric fluid dynamics,
  - [VULCAN][4] atmospheric chemistry.
